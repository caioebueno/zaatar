# Menus API

## Endpoints

`GET /api/menus`

- Returns all menus ordered by default first.
- Ensures the hardcoded default menu (`id: "default"`) exists.

`POST /api/menus`

Create a menu.

Payload:
- `name` (required string)
- `id` (optional string; autogenerated if omitted)
- `active` (optional boolean, default `true`)
- `isDefault` (optional boolean, default `false`)

`PATCH /api/menus/:menuId`

Update menu fields.

Payload (at least one required):
- `name?: string`
- `active?: boolean`
- `isDefault?: boolean`

If `isDefault=true`, all other menus are updated to `isDefault=false`.
